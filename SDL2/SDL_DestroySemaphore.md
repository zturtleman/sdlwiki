# SDL_DestroySemaphore

Destroy a semaphore.

## Syntax

```c
void SDL_DestroySemaphore(SDL_sem * sem);

```

## Function Parameters

|             |                          |
| ----------- | ------------------------ |
| **sem**     | the semaphore to destroy |

## Remarks

It is not safe to destroy a semaphore if there are threads currently
waiting on it.

## Version

This function is available since SDL 2.0.0.
